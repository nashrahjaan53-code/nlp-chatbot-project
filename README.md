#  Generative Friend AI Chatbot

An advanced, conversational Natural Language Processing (NLP) portfolio project leveraging Large Language Models (LLMs) to build a natural, context-aware digital friend.

##  Core Features
* **Generative Dialogue:** Powered by the Google Gemini API (`gemini-2.5-flash`) for dynamic, human-like conversations.
* **Persistent Session State:** Uses continuous execution memory to track context across multi-turn conversational streams.
* **Modern Web UI:** Beautiful, responsive web interface built cleanly via Streamlit.

##  Tech Stack & Architecture
* **Language:** Python
* **LLM Engine:** Google GenAI SDK
* **Frontend Dashboard:** Streamlit
* **Architecture:** Modular, production-ready package architecture (`src/`)
